📋 平台简介

CodeEdu 是一个功能完整的在线编程评测系统，专为国内网络环境优化。支持编程教学、算法竞赛、班级管理和个人练习等多种场景。

✅ 核心优势：国内网络友好、零编程要求、离线运行、部署简单、功能完整

🚀 部署指南

📦 单机部署

# 1. 下载项目文件
# 2. 双击"启动教学平台_优化版.bat"
# 3. 访问 http://localhost:5173

🌐 离线部署

# 1. 运行"创建离线部署包.bat"
# 2. 分发ZIP文件
# 3. 运行"一键安装.bat"
# 4. 启动系统
