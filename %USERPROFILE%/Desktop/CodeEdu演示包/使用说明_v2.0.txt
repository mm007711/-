# CodeEdu 在线编程教学平台 - 使用说明 v2.0

## 🎯 平台简介
CodeEdu 是一个功能完整的在线编程教学平台，专为计算机教育设计。平台集成了多班级管理、可视化数据分析、批量操作、专业报告生成等核心功能，并新增了演示模式。

## 📦 文件说明

### 1. 便携版软件包 (最新版)
- `CodeEdu教学平台_便携版_v2.0.zip` - 完整便携版软件（在桌面上）
- 包含所有最新功能：演示模式、多班级管理、批量操作等
- 解压后运行 `启动教学平台.bat` (Windows) 或 `./启动教学平台.sh` (Linux/Mac)

### 2. 演示包文件（本文件夹）
- `demo_class_data.json` - 演示数据（2个班级，20名学生，2场比赛，3道题目）
- `import_demo_data.py` - 数据导入工具
- `启动教学平台.bat` - Windows启动脚本
- `DEMO_PREPARATION_SUMMARY.md` - 详细演示指南
- `使用说明_v2.0.txt` - 本文件

## 🚀 快速开始

### 方法1：使用便携版（推荐）
1. 解压桌面上的 `CodeEdu教学平台_便携版_v2.0.zip`
2. 进入解压后的文件夹
3. 双击运行 `启动教学平台.bat` (Windows)
4. 或运行 `./启动教学平台.sh` (Linux/Mac)
5. 按照界面提示导入 `demo_class_data.json`

### 方法2：使用现有系统
1. 确保已安装 Python 3.8+
2. 运行 `python import_demo_data.py` 查看导入说明
3. 启动后端服务器：`python backend/simple_server.py`
4. 在浏览器中打开 `111.html`

## 🆘 常见问题

### Q: 启动时提示"未检测到Python"？
A: 请安装Python 3.8+，安装时勾选"Add Python to PATH"

### Q: 导入数据后不显示？
A: 刷新页面或重新打开111.html

### Q: 如何重置所有数据？
A: 清除浏览器缓存或使用隐私模式

### Q: 支持多少学生数据？
A: 理论上无限制，建议单班级不超过1000人
