平台简介

CodeEdu 是一个功能完整的在线编程评测系统，专为国内网络环境优化。支持编程教学、算法竞赛、班级管理和个人练习等多种场景。 无需编程知识即可使用，支持完全离线部署，特别适合学校、培训机构和个人的编程教学需求。

核心功能

国内网络优化

专为国内网络环境设计，使用国内镜像源，下载速度快，部署稳定。

  • npm淘宝镜像源
  • pip清华镜像源
  • 智能网络检测
  • 离线部署支持

多角色系统

完善的多角色权限管理系统，满足不同用户群体的需求。

  • 学生：提交代码、查看成绩
  • 教师：管理班级、布置作业
  • 管理员：系统管理、数据维护

智能评测系统

支持多种编程语言，自动评测代码，提供详细反馈。

  • 支持Python、Java、C++等
  • 实时评测和反馈
  • 详细的错误提示
  • 运行时间和内存统计

数据管理

强大的数据管理功能，支持批量导入导出。

  • CSV/Excel数据导入
  • JSON数据导出
  • 图形界面管理工具
  • 批量操作支持

部署指南

单机部署

适合个人学习和演示环境。

# 1. 下载项目文件
# 2. 双击"启动教学平台_优化版.bat"
# 3. 访问 http://localhost:5173

离线部署

适合无网络环境或批量部署。

# 1. 运行"创建离线部署包.bat"
# 2. 将生成的ZIP文件分发
# 3. 解压后运行"一键安装.bat"
# 4. 运行"启动教学平台.bat"
