🚀 部署指南

Netlify部署状态：✅ 已配置

本项目已配置Netlify部署，您可以通过以下方式部署：

1. GitHub部署

# 1. 将项目推送到GitHub
# 2. 登录Netlify
# 3. 选择"New site from Git"
# 4. 选择GitHub仓库
# 5. 自动部署完成

2. 手动部署

# 1. 安装Netlify CLI
npm install -g netlify-cli

# 2. 登录Netlify
netlify login

# 3. 初始化部署
netlify init

# 4. 部署网站
netlify deploy --prod

3. 本地运行

# 1. 安装依赖
cd frontend && npm install

# 2. 启动开发服务器
npm run dev

# 3. 访问地址
http://localhost:5173

📁 项目文件结构

  • netlify.toml - Netlify部署配置
  • frontend/ - Vue.js前端项目
  • backend/ - Python Flask后端API
  • 启动教学平台_优化版.bat - 一键启动脚本
  • 创建离线部署包.bat - 离线部署工具
