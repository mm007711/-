# CodeEdu 数据导入工具使用说明

## 概述

CodeEdu 数据导入工具是一个图形界面应用程序，专为不会编程的教师和管理员设计，用于轻松导入和管理教学平台的数据。该工具支持批量导入用户、题目、班级等数据，也支持手动添加和导出数据。

## 系统要求

1. **Python 3.6+** - 必须安装Python
2. **必要库** - 需要安装以下Python库：
   - `pandas` (用于Excel文件支持)
   - `openpyxl` (用于Excel文件支持)

安装命令：
```bash
pip install pandas openpyxl
```

## 快速开始

### 方法一：使用批处理文件启动（推荐）
1. 双击运行 `启动数据导入工具.bat`
2. 按照屏幕提示操作

### 方法二：使用命令行启动
```bash
python data_import_gui.py
```

## 工具界面介绍

工具包含四个主要选项卡：

### 1. 批量导入
- **导入用户**：从CSV或Excel文件批量导入用户
- **导入题目**：从CSV或Excel文件批量导入编程题目
- **导入班级**：从CSV或Excel文件批量导入班级

### 2. 手动添加
- **添加用户**：手动输入单个用户信息
- **添加题目**：手动输入单个题目信息

### 3. 数据导出
- **导出到JSON**：将数据导出为JSON格式
- **导出到CSV**：将数据导出为多个CSV文件

### 4. 系统状态
- **数据库状态**：查看数据库连接状态和各表记录数
- **操作日志**：查看所有操作的日志信息

## 文件格式说明

### 用户文件格式 (CSV/Excel)
```
username,password,name,role,email
student1,123456,张三,student,zhangsan@example.com
teacher1,123456,李老师,teacher,lilaoshi@example.com
admin1,123456,王管理员,admin,admin@example.com
```

**字段说明：**
- `username`：用户名（必填）
- `password`：密码（默认：123456）
- `name`：真实姓名
- `role`：角色（student/teacher/admin，默认：student）
- `email`：邮箱地址

### 题目文件格式 (CSV/Excel)
```
title,description,difficulty,score,time_limit,memory_limit,tags,is_public
两数之和,给定一个整数数组...,easy,10,1000,256,数组;哈希表,true
反转链表,反转一个单链表...,medium,15,2000,512,链表,true
```

**字段说明：**
- `title`：题目标题（必填）
- `description`：题目描述
- `difficulty`：难度（easy/medium/hard，默认：easy）
- `score`：分值（默认：10）
- `time_limit`：时间限制(ms)（默认：1000）
- `memory_limit`：内存限制(MB)（默认：256）
- `tags`：标签（用分号分隔）
- `is_public`：是否公开（true/false，默认：true）

### 班级文件格式 (CSV/Excel)
```
name,description,teacher_id
计算机科学2026,计算机科学专业班级,2
软件工程2026,软件工程专业班级,2
```

**字段说明：**
- `name`：班级名称（必填）
- `description`：班级描述
- `teacher_id`：教师ID（默认：2，对应test_teacher）

## 使用步骤

### 步骤1：准备数据文件
1. 使用Excel或文本编辑器创建数据文件
2. 按照上述格式填写数据
3. 保存为CSV或Excel格式

### 步骤2：启动工具
1. 双击 `启动数据导入工具.bat`
2. 等待工具启动

### 步骤3：检查数据库连接
1. 点击"检查数据库连接"按钮
2. 确认所有表都存在且连接正常

### 步骤4：导入数据
1. 切换到"批量导入"选项卡
2. 点击"浏览..."选择数据文件
3. 点击"导入用户"/"导入题目"/"导入班级"按钮
4. 查看导入结果和日志

### 步骤5：验证数据
1. 切换到"系统状态"选项卡
2. 点击"刷新状态"查看导入的数据统计

## 常见问题

### Q1: 工具启动失败，提示缺少Python
**A:** 请先安装Python 3.x，可以从 https://www.python.org/downloads/ 下载

### Q2: 导入Excel文件失败
**A:** 需要安装pandas和openpyxl库：
```bash
pip install pandas openpyxl
```

### Q3: 中文显示乱码
**A:** 确保数据文件使用UTF-8编码保存

### Q4: 用户已存在错误
**A:** 工具会自动跳过已存在的用户/题目/班级，不会覆盖现有数据

### Q5: 数据库文件不存在
**A:** 请确保CodeEdu系统已正确安装，数据库文件 `codeedu.db` 应位于同一目录

## 高级功能

### 1. 数据导出
- **JSON导出**：导出完整的数据结构，适合备份和迁移
- **CSV导出**：导出为多个CSV文件，适合在其他系统中使用

### 2. 手动添加
- 当需要添加少量数据时，使用手动添加功能更快捷
- 支持实时验证，避免重复数据

### 3. 批量操作
- 支持同时导入多个文件
- 支持批量导出所有数据

## 注意事项

1. **数据安全**：导入前建议备份现有数据库
2. **文件编码**：所有文件必须使用UTF-8编码
3. **字段格式**：严格按照规定的字段格式准备数据
4. **数据验证**：导入前建议先导出现有数据作为参考
5. **权限管理**：确保有足够的权限访问数据库文件

## 技术支持

如遇到问题，请检查：
1. Python版本是否为3.6+
2. 是否安装了必要的库（pandas, openpyxl）
3. 数据库文件是否存在且可访问
4. 数据文件格式是否正确

如需进一步帮助，请联系系统管理员。
